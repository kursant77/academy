# A+ Academy - O'quv Markazi

Professional o'quv markazi uchun to'liq funksionallik bilan websayt va admin panel.

## 🚀 Tezkor Boshlash

1. **Kerakli paketlarni o'rnating:**

   ```bash
   npm install
   ```

2. **Environment variables sozlang:**
   `.env` fayl yarating va quyidagilarni qo'shing:

   ```env
   VITE_SUPABASE_URL=your_project_url
   VITE_SUPABASE_ANON_KEY=your_anon_key
   ```

3. **Database sozlang:**
   - Supabase Dashboard → SQL Editor ga o'ting
   - **`COMPLETE_DATABASE_SETUP.sql`** faylini oching
   - Barcha SQL kodini nusxalab SQL Editor'ga yopishtiring
   - **Run** tugmasini bosing

4. **Loyihani ishga tushiring:**
   ```bash
   npm run dev
   ```
